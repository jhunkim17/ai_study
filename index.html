API 키와 학교 번호를 입력한 후 '급식 조회' 버튼을 눌러주세요.

사용 방법

  1. 나이스 교육정보 개방 포털에서 API 키를 발급받으세요.
  2. 발급받은 API 키를 입력하고 '저장' 버튼을 눌러주세요.
  3. 교육청 코드를 입력하세요. (강원도교육청: K10)
  4. 해당 교육청 소속 학교의 행정표준코드를 입력하세요.
  5. 조회하고 싶은 날짜를 선택하세요. (선택하지 않으면 오늘 날짜)
  6. '급식 조회' 버튼을 눌러 해당 날짜의 급식 정보를 확인하세요.
  7. '오늘 급식' 버튼으로 빠르게 오늘 급식을 확인할 수 있습니다.
참고:
  • 나이스 교육정보 개방 포털에서 나이스 API 발급이 가능합니다. 또한 나이스 교육정보 개방 포털 內 학교 기본 정보에서 교육청 코드와 학교 번호(행정표준코드를) 확인할 수 있습니다.
